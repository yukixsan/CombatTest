🌟 Overview
PVFX Core Shader 2.0 is a highly-optimized, stylized Unlit URP shader for real-time visual effects.
It expands upon version 1.0 with a new Motion Texture Layer, advanced UV distortion systems, and enhanced mobile-friendly GPU utilities.
Designed specifically for impact VFX, slash effects, dissolve animations, aura pulses, magical FX, projectiles, and stylized mobile effects, PVFX Core 2.0 is lightweight, modular, and fully controllable via shader properties.
This release includes:
•	Unlit_PV_VFX_Core2.0.shader — upgraded VFX shader with Motion Texture System.
•	PVFX_CustomTimeDriver.cs — per-instance timing driver (same pipeline as 1.0).

🎨 Base Surface Controls
Property	Description
_MainTex (RGBA)	Main texture for color and alpha.
_Color	Global tint color.
_MainTex_Scroll (XY)	UV scrolling (panning).
_MainTex_Rot	Rotation speed (around UV center).
_UseCoreColor	Enables core color injection.
_CoreColor / _CoreIntensity	Adds energy glow inside main color.
Great for: flowing fire, magic surfaces, aura, energy waves.
________________________________________



🔥 Dissolve System
Property	Description
_DissolveTex	Noise/gradient mask for dissolve.
_DissolveScroll / _DissolveRot	Animate dissolve pattern.
_UseDissolveTiming	Enables automatic dissolve timeline.
_DissolveStart / Duration	Controls dissolve animation in seconds.
_ReverseDissolve	Reverses dissolve direction.
_DissolveAmount	Manual dissolve (0–1).
_EdgeColor / _EdgeWidth	Controls glowing dissolve outline.
⚠️ Important
To ensure clean reset of dissolve when objects respawn, attach:
PVFX_CustomTimeDriver.cs
(drives per-instance _CustomTime)
________________________________________
💫 Rim Lighting
Property	Description
_UseRim	Enables rim light.
_RimColor	Color of rim glow.
_RimPower	Sharpness of rim falloff.
_RimIntensity	Brightness multiplier.
_RimMap	Optional rim mask.
Useful for: magical outlines, glowing edges, stylized highlights.
________________________________________



🌟 Bloom System
Property	Description
_BloomMap	Mask for bloom.
_BloomColor	Bloom color.
_BloomIntensity	Bloom power.
_BloomAffectRimOnly	Bloom applies only on rim if enabled.
Works well with Post-Processing "Bloom".
________________________________________
🌊 UV Warp Distortion
Property	Description
_UVWarpAmount	Distortion amplitude.
_UVWarpSpeed	Wave movement speed.
_UVWarpFrequency	Frequency of wave.
_UVWarpMode	0 = Sine mode, 1 = Zigzag mode.
Used for: heat distortion, fluid motion, magic ripple.
________________________________________
⚡ Global Distortion System
Property	Description
_UseGlobalDistort	Enables global distortion.
_DistortTex	Noise texture driving distortion.
_DistortAmount	Distortion strength.
_DistortTexXSpeed / YSpeed	Scrolling direction of noise.
_UseUvZAsSeed	Uses UV.z as seed value (per-vertex random).
_DistortSeed	Additional custom seed.
Great for chaotic energy, unstable magic, void FX, electricity.

🌐 Wave / Shake FX

•	_ShakeUvX, _ShakeUvY

✔ Radial Waves
•	_RoundWaveStrength

✔ Directional Wave
•	_WaveAmount
•	_WaveSpeed

Ideal for slashes, impacts, and magic surges.

🎥 Motion Texture Layer (Independent FX Layer)
✨ NEW in 2.0
The Motion Texture Layer is a second texture system that animates independently from the main UV.
Useful for: hit-flashes, sparks, moving lights, secondary masks, slash trails, projectile tip, etc.
🔹 Enable Motion Layer
Property	Description
_UseMotionTex	Enables the system.
_MotionTex	Texture used for motion layer.
_MotionColor	Tint + blending factor.
________________________________________
🏃 Motion: MOVE (A → B)
Property	Description
_MoveFrom	Starting offset (UV).
_MoveTo	Ending offset (UV).
_MoveStartTime	Delay before moving.
_MoveDuration	Time from A → B.
_Loop	Loop movement.
________________________________________
🔍 Motion: SCALE (from pivot)
Property	Description
_ScalePivot	Pivot of scaling (0–1 range).
_ScaleAmount	Final scaling amount.
_ScaleStartTime	Scale animation delay.
_ScaleDuration	Total scale time.
________________________________________
🔄 Motion: ROTATION (around pivot)
Property	Description
_RotationPivot	UV pivot for rotation.
_RotationAngle	Final angle in degrees.
_RotationStartTime	Delay before rotation.
_RotationDuration	Animation duration.
________________________________________
✔ Looping
All animation modes support looping if _Loop = 1.

🧠 How to Use
Step 1 — Create Material
Choose shader:
PVFX/Unlit_PV_VFX_Core2.0
Assign _MainTex and optional _MotionTex.
Tweak dissolve, UV warp, rim, bloom, and motion settings.
________________________________________
Step 2 — Add Custom Time Driver
For dissolve timing or any _CustomTime behavior, attach:
PVFX_CustomTimeDriver.cs
This ensures:
•	per-instance timing
•	dissolve resets correctly
•	clean animation when FX respawn

💡 Tips & Optimization
✔ For mobile:
•	Disable unused features (rim, bloom, warp, distort).
•	Keep Motion Texture small (256–512px).
•	Avoid using all distortions at once.
•	Prefer grayscale noise for dissolve and distort.
✔ For slashes/projectiles:
•	Set Move A→B
•	Add slight rotation with pivot
•	Add RoundWave pulse
•	Final bloom gives readability
✔ For impact FX:
•	Use MotionTex as white burst
•	Add strong CoreColor
•	Use RoundWaveStrength for shockwave
